# 🧬 Grant AI Monitor

Automated AI system to track relevant grants from Grants.gov.

## Features
- Filters for genomics, bioinformatics, infectious disease
- AI-based eligibility check
- Email alerts
- Runs daily via GitHub Actions

## Target Grants
- K99
- R21
- F32

## Avoids
- R01
- UM1
- U24

## Setup
1. Add OpenAI API key
2. Add email credentials
3. Enable GitHub Actions
